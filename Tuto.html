Lancer l’application Virtual Konsol For Live.

La fenêtre suivante apparaît:

Nous allons en profiter pour définir l’entrée MIDI contrôlant  les changement de mémoire.

En haut de la fenêtre, cliquer sur le menu déroulant et sélectionner le BUS IAC (messages provenants d’Ableton Live).


Cliquer en suite sur l’onglet «Masters»

En bas à gauche, nous trouvons une section d’information sur le fonctionnement du moteur audio du VKL.

De haut en bas:

-I/O VS (Input/Output Vector size), c’est la taille du tampon audio.

  1. -ON/OFF du moteur audio.

  2. -DSP ouvre la fenêtre des préférences Audio MAX/MSP

  3. -La fréquence échantillonnage.

  4. -Le pourcentage de ressources CPU utilisé

  5. -Le driver audio d’entrée (choisir Soundflower 16ch pour accueillir les signaux audio en provenance de Live).

  6. -Le driver audio de sortie (choisir le driver de votre carte audionumérique si celle ci est connectée à l’ordinateur, sinon choisir CoreAudio Built-in Output).


Maintenant, cliquer sur le bouton DSP, ce qui ouvre la fenêtre de audio de MAX

Nous y retrouvons la plupart des réglages effectués précédemment.

Cliquer sur le bouton I/O Mapping:

    Ici, il est important de configurer le Input Map. comme sur l’image.

    C’est que qui va permettre de router les signaux audio dans les routings de lecteurs.


    Le Output Map. dépend de votre configuration. Dans mon cas, j’utilise le CoreAudio interne de mon macbook (assis, au soleil sur la terrasse...), c’est pourquoi seules les deux premières sorties sont mappées. Dans le cas d’une interface audio-numérique avec plusieurs sorties, il sera nécessaire de mapper les sorties supplémentaires.


    Cette partie est importante dans le cas où les sorties audios ne sont pas consécutives sur votre carte son. Par exemple, les 8 premières sont des sorties audio analogique, puis 2 pour le casque, puis les deux autres pour le SPDIF, et pour les 8 suivantes le port ADAT que vous utilisez via un ADI-8.


Le setup devra ressembler à ça:

Vous pouvez maintenant fermer ces fenêtre et revenir à la page principale du VKL.

Il est temps d’activer le CoreAudio Audio:

Cliquer sur le menu déroulant et sélectionner ON. Le moniteur d’activité CPU doit tourner autour de 10%

Ne pas oublier d’ouvrir les tranches de Masters que vous utilisez:

    Tous les paramètres de cette section peuvent être mémoriser dans une mémoire.

    Les rappels de mémoires peuvent se faire à partir de Live.

    En dessous de chaque fader, un nom de sortie peut être entré. Il suffit de cliquer sur un nom en vert et de rentrer le nom du HP à l’aide du clavier. Les noms sont sauvegardés avec l’ensemble des paramètres dans les mémoires.

    Pour des réglages plus fin sur chaque tranche, un clic sur le bouton SEL ouvre la fenêtre :
